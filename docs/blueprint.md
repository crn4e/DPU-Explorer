# **App Name**: DPU Explorer

## Core Features:

- Interactive Map: Display interactive map of DPU campus with key locations marked.
- Location Details: Show details for each location (building, cafeteria, gym, etc.), including opening hours, current status (open/closed), and a brief description.
- Location Filtering: Enable filtering of locations based on type (e.g., academic buildings only, food services only).
- Admin Content Update: Allow admin users to update location details, opening hours, and status on a daily basis via a password-protected admin page.
- Location Announcements: Display up-to-date announcements from the university pertaining to specific locations
- AI Tour Guide: Generate a dynamic campus tour itinerary, adjusting waypoints or locations as necessary if an area is closed or if its opening hours do not match. Generative AI will act as a planning tool that considers the current daily conditions when offering tour suggestions.

## Style Guidelines:

- Primary color: Deep crimson (#800000) reflecting the university's prestige.
- Background color: Off-white (#FAF9F6) for a clean, readable experience.
- Accent color: Gold (#FFD700) to highlight key information and interactive elements, creating contrast and signaling importance.
- Font: 'Alegreya', a serif font for headings and body text, lending an intellectual and elegant feel. 
- Use clear, modern icons to represent different location types, aiding navigation and visual appeal.
- Prioritize a clean and intuitive layout with a focus on easy navigation and readability, ensuring users can quickly find the information they need.
- Incorporate subtle transitions and animations to enhance user engagement and provide visual feedback, such as when updating location statuses or loading map data.
